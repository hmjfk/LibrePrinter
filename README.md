# LibrePrinter
悪徳商法から開放された自由な印刷機
> [!Note]
> この装置はまだ構想段階で、まだ実際の開発は行われていない。
## 概要
この印刷機は、以下を理念として設計されています。
- 内部構造が理解しやすく、組み立てがしやすいこと。
- 一切のDriverを必要としないこと。OSに認識させたいときは任意で導入できること。
- 会社製品でよくある使用者を困惑させる挙動をしないこと。
